Research Projects

Project 1: Word Order in Russian and English

Description: This project analyzes the frequency and structure of word order in Russian and English using natural language processing (NLP) tools. The goal is to understand how syntactic structures differ in both languages and to explore their impact on translation accuracy.

Excerpt: By examining a corpus of texts, we analyze the word order in both languages and compare the frequency of different sentence structures. This study provides insights into how language models can handle syntactic differences.

GitHub Repository: Word Order Project - GitHub

Project 2: Eye-Tracking Simulation Without Eye-Tracking Devices

Description: This project focuses on simulating eye-tracking patterns for sentence processing in the absence of actual eye-tracking equipment. Using statistical modeling and machine learning, the study simulates what eye movements might look like during reading.

Excerpt: The study uses eye-tracking data simulations based on syntactic and semantic cues to predict how readers process word order. The findings offer an innovative approach to understanding reading comprehension.

GitHub Repository: Eye-Tracking Simulation Project - GitHub

Project 3: Language Models and Word Order in Machine Translation

Description: This project explores how language models, especially those used for machine translation, handle different word orders in English and Russian. The goal is to optimize machine translation systems by focusing on syntactic accuracy.

Excerpt: We analyze the challenges that machine translation faces when dealing with sentences that have non-standard word orders. The study helps improve translation accuracy for languages with flexible syntax.

GitHub Repository: Machine Translation Project - GitHub
